# 🧮 Basic Calculator

A simple and responsive calculator built using **HTML, CSS, and JavaScript**. This project performs basic arithmetic operations and demonstrates fundamental JavaScript concepts such as DOM manipulation, event handling, and functions.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖ Multiplication
- ➗ Division
- 🔢 Decimal number support
- ⌫ Backspace functionality
- 🗑 Clear display button
- ⌨ Keyboard input support
- 🚫 Prevents consecutive operators
- 🎨 Responsive and clean user interface

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📂 Project Structure

```
calculator/
│── calculator.html
│── calculator.css
│── calculator.js
│── README.md
```

## 🚀 How to Run

1. Download or clone the repository.
2. Open `calculator.html` in your web browser.
3. Start performing calculations.
